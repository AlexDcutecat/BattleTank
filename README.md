# BattleTank
fc坦克大战python复刻v0.1.1                                                                     
运行Main.py进入游戏(需要python3的运行环境)，w和s键选择模式，Backspace键开始游戏(亦可用于游戏内暂停)        
进入游戏后P1玩家使用w，s，a，d控制移动，j控制攻击;
P2玩家使用↑，↓，←，→控制移动，/控制攻击                                    
游戏中使用数字1复活                                  
目前只有一关，且尚未做通关后结算动画                           
创作模式亦可用于地图制作(map文件夹内)
